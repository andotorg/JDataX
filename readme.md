### JDataX是整合DataX的一个系统

#### 目前功能：
  批量处理同步数据库， 通过读取数据库模板文件进行批量创建json文件
  SQL文件：在项目根目录 jdatax.sql  Mysql数据库
  
#### 计划
1. 2019-10-01 V1.1实现 加入实时显示进度
    例如 目前同步到那张表，同步是否成功，在页面上显示datax控制台打印的信息

2. 2019-12-01 V1.2实现 加入定时任务调度，自动同步
    例如 设置定时任务，自动同步,什么时间开始，什么时间定时触发
